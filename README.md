# Responsive Webpage Layout

## Description
This project demonstrates a responsive webpage layout designed using **HTML and CSS**. The webpage is divided into four main sections: **Header**, **Sidebar**, **Main Content**, and **Footer**. The layout adapts smoothly to different screen sizes to provide an optimal user experience on both desktop and mobile devices.

## Features
- Structured layout with four sections:
  - Header
  - Sidebar
  - Main Content
  - Footer
- Responsive design using **CSS media queries**
- **Grid layout** for desktop screens
- **Stacked layout** for mobile devices
- Clean and simple UI suitable for beginners

## Technologies Used
- HTML5
- CSS3 (Flexbox / Grid)
- Media Queries

## Responsive Behavior
- **Desktop view:** Uses a grid-based layout with sidebar and main content side by side
- **Mobile view:** Switches to a stacked layout for better readability

## How to Run
1. Download or clone the project files
2. Open the `index.html` file in any modern web browser
3. Resize the browser window to see the responsive behavior
